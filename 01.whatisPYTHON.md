# Python이란?

Python은 쉽고 간단한 문법을 가진 프로그래밍 언어이다.  
처음 배우는 사람도 비교적 빠르게 익힐 수 있어서 입문용 언어로 많이 사용된다.

특히 웹 개발, 인공지능, 데이터 분석, 자동화, 해킹/보안 분야까지 매우 넓게 사용된다.  
마치 "만능 공구상자" 같은 언어라고 볼 수 있다.

## Python의 특징

- 문법이 간단하다
- 코드가 짧고 읽기 쉽다
- 다양한 분야에 사용할 수 있다
- 사용할 수 있는 라이브러리가 많다
- 초보자도 배우기 쉽다

## Python은 어디에 사용될까?

| 분야 | 예시 |
| --- | --- |
| 웹 개발 | 웹사이트 만들기 |
| 데이터 분석 | 데이터 정리, 그래프 만들기 |
| 인공지능 | AI, 머신러닝 |
| 자동화 | 반복 작업 자동으로 하기 |
| 보안 / 해킹 | 취약점 분석, 자동 공격 도구 제작 |

## 보안 분야에서 Python을 배우는 이유

보안 분야에서는 반복 작업을 자동으로 하거나, 여러 대상을 빠르게 검사해야 하는 경우가 많다.

Python을 사용하면 다음과 같은 프로그램을 만들 수 있다.

- 여러 사이트에 자동으로 요청 보내기
- 특정 문자열 찾기
- 파일 자동 분석하기
- 간단한 해킹 도구 만들기
- 네트워크 정보 수집하기
